# DecodeLabs AI Internship Tasks

This repository contains 4 AI/ML projects completed as part of the DecodeLabs Internship program.  
Each task demonstrates a different core concept in Artificial Intelligence and Machine Learning.

---

## Project Structure
decodelabs_tasks/

- Task 1 - Rule Based Chatbot  
- Task 2 - Titanic Classification Model  
- Task 3 - Movie Recommendation System  
- Task 4 - Image Recognition (Cat vs Dog)


---

## Task 1: Rule-Based AI Chatbot

- Built a simple chatbot using Python
- Uses **if-else logic and dictionary-based responses**
- Handles greetings, questions, and exit commands

### Skills:
- Control flow
- Logic building
- Basic AI concepts

---

## Task 2: Titanic Classification Model

- Machine learning model using **Decision Tree Classifier**
- Predicts passenger survival
- Uses dataset preprocessing and feature selection

### Skills:
- Supervised learning
- Data cleaning
- Model training & evaluation

---

## Task 3: Movie Recommendation System

- Content-based recommendation system
- Uses **cosine similarity**
- Recommends movies based on user preferences

### Skills:
- NLP preprocessing
- Feature engineering (tags)
- Similarity measurement

---

## Task 4: Image Recognition (Cat vs Dog)

- CNN-based image classification model using TensorFlow/Keras
- Classifies images as Cat or Dog
- Includes preprocessing, training, and prediction

### Dataset:
- Dogs vs Cats dataset (Kaggle)
- Downloaded using Kaggle API

### Skills:
- Deep learning (CNN)
- Computer vision
- Model evaluation

- Python
- TensorFlow / Keras
- Scikit-learn
- OpenCV
- Pandas / NumPy
